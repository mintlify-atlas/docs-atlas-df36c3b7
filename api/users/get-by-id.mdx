---
title: 'Get User by ID'
description: 'Retrieve a specific user by their unique identifier'
api: 'GET /api/v1/users/{id}'
---

## Endpoint

```
GET /api/v1/users/{id}
```

Returns the details of a specific user identified by their unique UUID.

## Path Parameters

<ParamField path="id" type="UUID" required>
  Unique user identifier.
  
  **Example:** `0f4df2de-fffb-4a24-9891-381ecf4f0f87`
</ParamField>

## Response

<ResponseField name="id" type="UUID">
  Unique identifier for the user.
  
  **Example:** `0f4df2de-fffb-4a24-9891-381ecf4f0f87`
</ResponseField>

<ResponseField name="username" type="string">
  The user's username.
  
  **Example:** `jdoe`
</ResponseField>

<ResponseField name="email" type="string">
  The user's email address.
  
  **Example:** `jdoe@example.com`
</ResponseField>

<ResponseField name="firstName" type="string">
  The user's first name.
  
  **Example:** `John`
</ResponseField>

<ResponseField name="lastName" type="string">
  The user's last name.
  
  **Example:** `Doe`
</ResponseField>

<ResponseField name="role" type="string">
  The user's role.
  
  **Example:** `USER`
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the user was created.
  
  **Example:** `2024-01-15T10:30:00`
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when the user was last updated.
  
  **Example:** `2024-01-15T10:30:00`
</ResponseField>

<ResponseField name="active" type="boolean">
  Whether the user account is active.
  
  **Example:** `true`
</ResponseField>

## Status Codes

<ResponseField name="200" type="OK">
  User retrieved successfully
</ResponseField>

<ResponseField name="404" type="Not Found">
  User with the specified ID does not exist
</ResponseField>

<ResponseField name="500" type="Internal Server Error">
  An unexpected error occurred
</ResponseField>

## Example Request

```bash
curl http://localhost:8080/api/v1/users/0f4df2de-fffb-4a24-9891-381ecf4f0f87
```
